# Cortex Design Goals

## Core Principles

1. **Optimization over abstraction** - Built for production performance
2. **Single-file deployment** - Zero dependency hell
3. **Framework compatibility** - Works with CrewAI, AutoGen, LangGraph
4. **Mojo-native speed** - 35000x faster than Python coordination

## Non-Goals

- Not a multi-agent framework (use existing frameworks)
- Not a model format (use GGUF, ONNX, etc.)
- Not a general-purpose container (use Docker for that)
